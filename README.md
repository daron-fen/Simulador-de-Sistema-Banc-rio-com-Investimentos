# Simulador-de-Sistema-Banc-rio-com-Investimentos
Simulador de Sistema Bancário com Investimentos desenvolvido em Python e Django. O projeto simula operações financeiras essenciais, como depósitos, saques e transferências, além de um módulo de investimentos em renda fixa e variável. Frontend interativo em HTML, CSS e JavaScript, integrado a um backend robusto e seguro.
